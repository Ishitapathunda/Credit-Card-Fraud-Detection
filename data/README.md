# Data Directory

Place the `creditcard.csv` file from Kaggle here.

## Download Instructions

1. Go to [Kaggle - Credit Card Fraud Detection](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)
2. Download the `creditcard.csv` file
3. Place it in this directory: `data/creditcard.csv`

## Dataset Information

- **File**: creditcard.csv
- **Size**: ~150 MB
- **Rows**: 284,807 transactions
- **Columns**: 31 (Time, V1-V28, Amount, Class)
